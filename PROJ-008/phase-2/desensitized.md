```

              ┌────────────────┼────────────────┐
              ▼                ▼                ▼
       ┌──────────┐     ┌──────────┐     ┌──────────┐
       │ 闲聊通道  │     │ API通道  │     │ Web通道  │
       │ (免费)    │     │ (按量)   │     │ (0 token)│
       │ 3B本地回答 │     │ 代码/推理│     │ 新闻/搜索 │
       │ 0成本     │     │ 文案/生图│     │ Playwright│

                           ¥按量            ¥0
```

- **API通道**: 复杂代码生成、推理、文案、生图 → 走DeepSeek/豆包API
- **Web通道**: 信息检索、新闻查询、事实核实 → Playwright自动化搜索, 0 token
- **闲聊通道**: 日常对话、创作 → 3B本地回答, 0成本
- **兜底规则**: 任务不明确 → 默认走API通道，宁可花钱不可出错

### 1.5 四层温度记忆模型 + 记忆弹出机制（基于noah-core + Cerebella）

**记忆弹出机制**（项目总索引: 待立项 → 本阶段正式设计）:

```
记忆 → 温度检测
    │
    ├─ 🔥 热记忆 (<7天 / 高频访问)
    │    └─ 完整内容 → 0.5B本地SQLite → 弹出响应 (~0.2s)
    │
    ├─ 🌤 温记忆 (7-30天 / 中频)
    │    └─ 压缩摘要 → exact_info + memory_store → 弹出 (~5ms)
    │
    ├─ ❄️ 冷记忆 (30-90天 / 低频)
    │    └─ 关键词+摘要 → pgvector语义检索 → 弹出 (~50ms-2s)
    │
    └─ 🏛 仓库记忆 (>90天 / 归档)
         └─ 一句话指针+路径 → MD档案 → 按需加载 (~500ms+)
```

**温度迁移规则**:

| 迁移方向 | 阈值 | 动作 |
|:---------|:----:|:------|
| 热→温 | 7天无访问 | 移文件到普通记忆目录 |
| 温→冷 | 30天无访问 | 压缩为摘要+关键参数 |
| 冷→仓库 | 90天无访问 | 一句话+标签+路径指针 |
| 任意→🔥热弹出 | 被用户访问/关联触发 | 恢复完整内容到HOT缓存 |

### 1.6 三阶段技能内化引擎（基于Cerebella Architecture）

这是Cerebella最核心的进化逻辑——从依赖检索到零检索本地闭环：

| 阶段 | 触发条件 | 行为 | 延迟 | 调用成本 |
|:----:|:---------|:-----|:----:|:--------:|
| **① 索引导航** | 知识库有相关文档 | 检索pgvector→返回文件路径+摘要 | ~50ms-2s | 需调用嵌入API |
| **② 技能调度** | 同类方案积累≥10条 | 从skills库调取完整技能直接使用 | ~0.5s | 本地零成本 |
| **③ 知识内化** | 同一技能命中≥30次 | 0.5B直接从经验库匹配，零检索 | ~0.2s | 0 token |

**阶段跃迁配置**:
```
stage1_to_stage2_threshold: 10  # 积累10条同类有效方案→升为技能
stage2_to_stage3_threshold: 30  # 同一技能命中30次→内化为本能
```

### 2.1 🧠 神经学专家视角

#### 2.1.1 六层映射的神经学合理性

#### 2.1.2 四层记忆温度的神经学类比

| 记忆温度 | 神经学结构 | 功能 | 诺亚实现 |
|:--------:|:----------|:-----|:---------|
| 🔥 **热记忆** | 工作记忆(前额叶) | 当前对话、活跃项目 | 3B上下文(~32K tokens) |
| 🌤 **温记忆** | 近期情景(海马体) | 最近7-30天经历 | exact_info+memory_store |
| ❄️ **冷记忆** | 远期内隐(皮层) | 30-90天压缩知识 | pgvector矢量库 |

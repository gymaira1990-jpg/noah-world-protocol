## 3. 四层温度记忆模型 + 记忆弹出机制

### 温度分层

```
记忆 → 温度检测
    │
    ├─ 🔥 热记忆 (<7天 / 高频访问)
    │    └─ 完整内容 → 0.5B本地SQLite → 弹出响应 (~0.2s)
    │
    ├─ 🌤 温记忆 (7-30天 / 中频)
    │    └─ 压缩摘要 → exact_info + memory_store → 弹出 (~5ms)
    │
    ├─ ❄️ 冷记忆 (30-90天 / 低频)
    │    └─ 关键词+摘要 → pgvector语义检索 → 弹出 (~50ms-2s)
    │
    └─ 🏛 仓库记忆 (>90天 / 归档)
         └─ 一句话指针+路径 → MD档案 → 按需加载 (~500ms+)
```

### 温度迁移规则

| 迁移方向 | 阈值 | 动作 |
|:---------|:----:|:------|
| 热→温 | 7天无访问 | 移文件到普通记忆目录 |
| 温→冷 | 30天无访问 | 压缩为摘要+关键参数 |
| 冷→仓库 | 90天无访问 | 一句话+标签+路径指针 |
| 任意→🔥热弹出 | 被用户访问/关联触发 | 恢复完整内容到HOT缓存 |

### 弹出优先级

HOT缓存(0成本) > 精确匹配(1ms) > 情景记忆(5ms) > 语义搜索(50ms-2s) > 归档(500ms+)

**弹出流程**:
1. 检查0.5B本地HOT缓存(≤10条) — 命中→~0.2s
2. exact_info精确匹配(键值对) — 命中→~1ms
3. memory_store标签过滤(近期情景) — 命中→~5ms
4. 豆包嵌入→pgvector语义搜索(冷知识) — 命中→~50ms-2s
5. MD档案全文索引(归档) — 命中→~500ms+

## 4. 三阶段技能内化引擎

从依赖检索到零检索本地闭环的进化逻辑：

| 阶段 | 触发条件 | 行为 | 延迟 | 调用成本 |
|:----:|:---------|:-----|:----:|:--------:|
| **① 索引导航** | 知识库有相关文档 | 检索pgvector→返回文件路径+摘要 | ~50ms-2s | 需调用嵌入API |
| **② 技能调度** | 同类方案积累≥10条 | 从skills库调取完整技能直接使用 | ~0.5s | 本地零成本 |
| **③ 知识内化** | 同一技能命中≥30次 | 0.5B直接从经验库匹配，零检索 | ~0.2s | 0 token |

**阶段跃迁配置**:
```
stage1_to_stage2_threshold: 10  # 积累10条同类有效方案→升为技能
stage2_to_stage3_threshold: 30  # 同一技能命中30次→内化为本能
```

### AB方案仲裁 + 三证训练闭环

| 步骤 | 操作 | 详情 |
|:----:|:-----|:------|
| 1 | AB双轨生成 | A轨: 本地脑(3B/0.5B/1.5B) → 方案A；B轨: 工厂层(DeepSeek/豆包/Web) → 方案B |
| 2 | 大模型评分仲裁 | 可行性30%/完整性30%/安全性20%/效率20%；本地优先 |
| 3 | 用户决策门闸 | 用户: 通过/修改/拒绝 |
| 4 | 执行+有效性验证 | 确认结果→标记有效/失败 |
| 5 | 技能转化+三阶段内化 | SKILL.md自动生成→lightweight-db→三证训练 |

**三证训练**:
- **证一量证**: 同类方案≥10条→触发蒸馏
- **证二质证**: 大模型评分≥80/100才通过
- **证三稳证**: 回归测试无退化→写入小模型权重
